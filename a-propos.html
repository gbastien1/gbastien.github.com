Un peu sur moi

Je suis nouvellement finissante au Baccalauréat en informatique de l'UQAC. Mon expérience s'est pour l'instant dirigée vers les langages de base comme HTML, CSS et Javascript, puis vers certains frameworks web comme jQuery, Twitter Bootstrap et Django. Ceci dit, mon but est de poursuivre mes apprentissages vers le langage Javascript et ses librairies et frameworks les plus utilisés comme AngularJS et React. Javascript étant le langage web de base, il m'est important de mieux le connaître. Je réalise dernièrement des petits projets avec Angular dans le but de me familiariser avec ce framework. Je reste toutefois intéressée par la programmation de jeux vidéos en C# avec Unity 5. Je réalise donc aussi certains projets dans ce domaine.

Je suis présentement en rechercne d'emploi dans le domaine du web front-end, mais «fullstack» me convient aussi.
