--- title: Carte interactive lang: fr ref: carte-interactive type: Projet professionel mydate: 2016-05-09 layout: project image: /assets/img/projets/carte-interactive.png external-url: http://www.uqac.ca/cartebdi link-title: "www.uqac.ca/cartebdi (user: visiteur; mdp: cartebdi)" Technologies: - Python / Django - Google Maps API - HTML / CSS - Javascript - Bootstrap 3 - JSON - Microsoft Excel (openpyxl) - Heroku --- Projet réalisé pour un cours Stage/projet à l'UQAC dans le cadre d'un contrat réel avec un client. Il s'agit d'une carte interactive de la France où seront pointées des écoles à visiter pour le recrutement d'écoles étrangères par l'UQAC. L'application devra toujours être à jour avec un fichier Excel qui contient les mêmes données que la base de données. Elle utilise Postgresql et est hébergée gratuitement sur Heroku.
